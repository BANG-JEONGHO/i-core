# 사용자 페르소나

## Persona 1: 김사원 (사내 직원 — 주 사용자)

| 항목 | 내용 |
|---|---|
| **이름** | 김사원 |
| **역할** | 교육기업 직원 (사업기획팀, 교육운영팀 등) |
| **목표** | 나라장터 과업지시서에 맞는 최적 강사를 빠르게 찾아 제안서 작성에 활용 |
| **기술 수준** | 일반 사무직 (IT 비전문가, 웹 서비스 사용에 익숙) |
| **주요 Pain Point** | 수동으로 108명 강사 이력서를 하나씩 검토하며 적합 강사를 찾는 데 시간이 많이 소요됨 |
| **사용 빈도** | 주 2~5회 (과업지시서 공고에 따라 변동) |
| **기대 결과** | 과업지시서 업로드 → 몇 초 내 매칭 결과 확인 → 강사 선정 근거 확보 |

### 사용 시나리오
1. 나라장터에서 과업지시서 다운로드
2. 플랫폼에 과업지시서 업로드
3. 자동 파싱된 평가기준/참여자격 확인
4. 매칭 결과 목록에서 적합 강사 선택
5. 매칭 근거를 참고하여 제안서 작성
