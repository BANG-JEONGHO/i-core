# 유저 스토리

형식: "~로서, ~하고 싶다, ~하기 위해"
우선순위: P1(1단계 핵심) / P2(1단계 부가) / P3(2단계) / P4(3단계)

---

## Epic 1: 인증

### US-01: 로그인
**As a** 사용자, **I want to** 회사 계정으로 로그인하고 싶다, **so that** 플랫폼에 접근할 수 있다.
- **우선순위**: P2
- **AC**: 올바른 ID/비밀번호 입력 시 대시보드로 이동한다.

### US-02: 로그아웃
**As a** 사용자, **I want to** 로그아웃하고 싶다, **so that** 세션을 안전하게 종료할 수 있다.
- **우선순위**: P2
- **AC**: 로그아웃 클릭 시 로그인 페이지로 이동하며 세션이 무효화된다.

### US-03: 회원가입
**As a** 사용자, **I want to** 회사 계정을 등록하고 싶다, **so that** 플랫폼을 사용할 수 있다.
- **우선순위**: P2
- **AC**: ID/비밀번호/이름 입력 후 계정이 생성된다.

---

## Epic 2: 강사 데이터 관리

### US-04: 강사 데이터 일괄 업로드
**As a** 사용자, **I want to** Excel/CSV 파일로 강사 이력서 데이터를 일괄 업로드하고 싶다, **so that** 매칭에 사용할 강사 풀을 구축할 수 있다.
- **우선순위**: P2
- **AC**: Excel/CSV 업로드 시 강사 데이터가 시스템에 저장되고 건수가 표시된다.

### US-05: 강사 목록 조회
**As a** 사용자, **I want to** 등록된 강사 목록을 조회하고 싶다, **so that** 현재 보유한 강사 풀을 파악할 수 있다.
- **우선순위**: P2
- **AC**: 강사 목록이 이름/전문분야 기준으로 정렬되어 표시된다.

### US-06: 강사 상세 정보 조회
**As a** 사용자, **I want to** 개별 강사의 상세 정보를 확인하고 싶다, **so that** 강사의 자격과 경력을 파악할 수 있다.
- **우선순위**: P2
- **AC**: 강사 클릭 시 전문분야, 경력, 자격증, 교육과목 등 상세 정보가 표시된다.

### US-07: 강사 정보 수정
**As a** 사용자, **I want to** 강사 정보를 수정하고 싶다, **so that** 최신 이력을 반영할 수 있다.
- **우선순위**: P2
- **AC**: 수정 후 저장 시 변경사항이 즉시 반영된다.

### US-08: 강사 삭제
**As a** 사용자, **I want to** 퇴사/비활성 강사를 삭제하고 싶다, **so that** 매칭 풀을 최신 상태로 유지할 수 있다.
- **우선순위**: P2
- **AC**: 삭제 확인 후 해당 강사가 목록과 매칭 대상에서 제거된다.

### US-09: 강사 검색/필터
**As a** 사용자, **I want to** 전문분야, 자격증 등으로 강사를 검색/필터하고 싶다, **so that** 원하는 강사를 빠르게 찾을 수 있다.
- **우선순위**: P2
- **AC**: 검색어/필터 적용 시 조건에 맞는 강사만 표시된다.

## Epic 3: 과업지시서 업로드 및 파싱

### US-10: 과업지시서 파일 업로드
**As a** 사용자, **I want to** 나라장터에서 받은 과업지시서 파일을 업로드하고 싶다, **so that** 자동 매칭을 시작할 수 있다.
- **우선순위**: P1
- **AC**: PDF/HWP/Word 파일 업로드 시 파일이 저장되고 파싱이 시작된다.

### US-11: 과업지시서 자동 파싱 — 참여자격 추출
**As a** 사용자, **I want to** 과업지시서에서 참여자격(신청자격)이 자동 추출되길 원한다, **so that** 수동으로 문서를 읽지 않아도 된다.
- **우선순위**: P1
- **AC**: 업로드 완료 후 추출된 참여자격 항목이 화면에 표시된다.

### US-12: 과업지시서 자동 파싱 — 평가기준 추출
**As a** 사용자, **I want to** 과업지시서에서 평가기준이 자동 추출되길 원한다, **so that** 매칭 기준을 파악할 수 있다.
- **우선순위**: P1
- **AC**: 업로드 완료 후 추출된 평가기준(배점 포함)이 화면에 표시된다.

### US-13: 파싱 결과 확인 및 수정
**As a** 사용자, **I want to** 자동 파싱된 결과를 확인하고 필요시 수정하고 싶다, **so that** 파싱 오류를 보정할 수 있다.
- **우선순위**: P1
- **AC**: 추출된 정보를 편집할 수 있고, 수정 후 매칭에 반영된다.

### US-14: 과업지시서 이력 조회
**As a** 사용자, **I want to** 이전에 업로드한 과업지시서 목록을 조회하고 싶다, **so that** 과거 매칭 내역을 참고할 수 있다.
- **우선순위**: P2
- **AC**: 업로드 일시 순으로 과업지시서 이력이 표시된다.

## Epic 4: 강사 매칭

### US-15: 키워드 기반 매칭 실행 (1단계)
**As a** 사용자, **I want to** 파싱된 참여자격/평가기준으로 강사 매칭을 실행하고 싶다, **so that** 조건에 맞는 강사를 찾을 수 있다.
- **우선순위**: P1
- **AC**: 매칭 실행 시 점수 기반으로 정렬된 강사 목록이 표시된다.

### US-16: 규칙 기반 가중치 점수 매칭 (1단계)
**As a** 사용자, **I want to** 전문분야 일치와 경력/자격증을 가중치로 반영한 매칭 점수를 보고 싶다, **so that** 종합적인 적합도를 파악할 수 있다.
- **우선순위**: P1
- **AC**: 각 강사에 종합 점수가 표시되며, 전문분야/경력/자격증 항목별 배점이 확인된다.

### US-17: AI 임베딩 기반 매칭 (2단계)
**As a** 사용자, **I want to** 의미 기반 유사도로 더 정확한 매칭 결과를 받고 싶다, **so that** 키워드 불일치 시에도 적합 강사를 발견할 수 있다.
- **우선순위**: P3
- **AC**: 임베딩 유사도 점수가 기존 규칙 점수와 결합되어 표시된다.

### US-18: Knowledge Graph 기반 매칭 (3단계)
**As a** 사용자, **I want to** 강사-기술-자격증 관계를 활용한 고도화된 매칭을 받고 싶다, **so that** 숨겨진 적합성도 발견할 수 있다.
- **우선순위**: P4
- **AC**: Knowledge Graph 관계 기반 추천이 결과에 포함된다.

### US-19: AI Agent 매칭 검증 (3단계)
**As a** 사용자, **I want to** AI Agent가 매칭 결과를 검증하고 설명해주길 원한다, **so that** 매칭 신뢰도를 높일 수 있다.
- **우선순위**: P4
- **AC**: 매칭 결과에 AI Agent의 검증 의견과 추천 이유가 포함된다.

---

## Epic 5: 매칭 결과 확인

### US-20: 매칭 결과 점수 목록 보기
**As a** 사용자, **I want to** 매칭된 강사를 점수 순으로 정렬된 목록으로 보고 싶다, **so that** 직관적으로 최적 강사를 파악할 수 있다.
- **우선순위**: P1
- **AC**: 전체 강사가 매칭 점수 내림차순으로 표시된다.

### US-21: 강사 매칭 상세 카드 보기
**As a** 사용자, **I want to** 강사를 클릭하면 카드 형식으로 매칭 근거를 보고 싶다, **so that** 왜 이 강사가 추천되었는지 이해할 수 있다.
- **우선순위**: P1
- **AC**: 강사 클릭 시 항목별 매칭 점수와 근거가 카드로 표시된다.

### US-22: 매칭 결과 내 강사 비교
**As a** 사용자, **I want to** 여러 강사의 매칭 결과를 비교하고 싶다, **so that** 최종 강사 선정을 쉽게 할 수 있다.
- **우선순위**: P2
- **AC**: 2~3명 강사를 선택하면 비교 뷰가 표시된다.

---

## Epic 6: 대시보드 및 이력

### US-23: 대시보드 — 최근 매칭 이력
**As a** 사용자, **I want to** 대시보드에서 최근 매칭 이력을 확인하고 싶다, **so that** 업무 현황을 한눈에 파악할 수 있다.
- **우선순위**: P2
- **AC**: 최근 매칭 5~10건이 날짜/과업지시서명과 함께 표시된다.

### US-24: 대시보드 — 강사 통계
**As a** 사용자, **I want to** 전문분야별 강사 분포를 확인하고 싶다, **so that** 강사 풀의 강/약점을 파악할 수 있다.
- **우선순위**: P2
- **AC**: 전문분야별 강사 수가 차트/그래프로 표시된다.

|---|---|---|
| **P1 (1단계 핵심)** | US-10, 11, 12, 13, 15, 16, 20, 21 | 과업지시서 업로드 → 파싱 → 매칭 → 결과 확인 |
| **P2 (1단계 부가)** | US-01~09, 14, 22, 23, 24 | 인증, 강사 관리, 대시보드 |

| **P4 (3단계)** | US-18, 19 | Knowledge Graph + AI Agent |
