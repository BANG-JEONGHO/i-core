# 비즈니스 규칙 — unit-matching-core

## Document Parser 규칙

### BR-P01: 파일 형식 제한
- 허용 형식: PDF, HWP, DOCX만 지원
- 최대 파일 크기: 50MB
- 검증: 확장자 + Magic bytes 이중 검증
- 위반 시: ParseError("지원하지 않는 파일 형식입니다")

### BR-P02: 텍스트 추출 실패 처리
- 파일이 손상되었거나 비밀번호 보호된 경우 → ParseError
- 이미지 전용 PDF (스캔본) → ParseError("텍스트를 추출할 수 없습니다. OCR이 필요한 문서입니다")
- 빈 문서 → ParseError("빈 문서입니다")

### BR-P03: 섹션 식별 규칙
- 섹션 헤더 패턴 (대소문자 무관):
  - 참여자격: `참여자격|신청자격|입찰참가자격|참가자격|응찰자격`
  - 평가기준: `평가기준|기술평가|배점|심사기준|적격심사`
- 섹션이 식별되지 않으면: 전체 텍스트를 반환하고 사용자에게 수동 지정 요청
- 동일 유형 섹션이 여러 개 발견되면: 모두 포함

### BR-P04: 키워드 추출 규칙
- 1글자 키워드는 제외 (조사, 접속사 등)
- 불용어 제거 (것, 등, 및, 또는, 이상, 이하 등)
- 복합 명사 유지 ("정보처리기사", "인공지능" 등)
- 중복 키워드 제거

### BR-P05: 필수/우대 구분 규칙
- 필수 키워드: "필수", "반드시", "갖추어야", "해야 한다", "충족"
- 우대 키워드: "우대", "가점", "가산", "바람직", "선호"
- 기본값: 키워드 없으면 is_mandatory = True

## Matching Engine 규칙

### BR-M01: 점수 범위
- 모든 점수는 0~100 범위
- 소수점 첫째자리까지 표시 (반올림)
- 음수 점수 불가

### BR-M02: 가중치 기본값
- 키워드(전문분야) 매칭: 40점 (40%)
- 자격증/학력 매칭: 30점 (30%)
- 경력 매칭: 30점 (30%)
- 합계: 항상 100점

### BR-M03: 키워드 매칭 규칙
- 정확한 문자열 일치 (대소문자 무관)
- 부분 일치 허용: "정보처리" → "정보처리기사" 매칭됨
- 동의어 처리: 기본 동의어 사전 활용 (예: "AI" = "인공지능" = "artificial intelligence")
- 과업지시서 키워드가 0개인 경우: 키워드 점수 = 0

### BR-M04: 자격 매칭 규칙
- 필수 자격만 점수에 반영 (우대 자격은 가산점으로 처리)
- 자격증 매칭: 이름 정확 일치 또는 등급 상위 호환
  - 예: "정보처리기사" 요구 → "정보처리기술사" 보유 = 충족
- 학력 매칭: 동등 이상이면 충족
  - 학사 요구 → 석사/박사 보유 = 충족
- 필수 자격이 0개인 경우: 자격 점수 = 만점(30)

### BR-M05: 경력 매칭 규칙
- 요구 경력 이상이면 만점
- 요구 경력 미달이면 비례 점수: (보유경력 / 요구경력) * 30
- 경력 요구가 명시되지 않으면: 경력 점수 = 만점(30)
- 최소 경력 0년이면: 모든 강사 만점

### BR-M06: 동점 처리
- 동일 점수인 경우 정렬 기준:
  1. 경력 연수 높은 순
  2. 자격증 수 많은 순
  3. 이름 가나다순

### BR-M07: 최소 점수 임계값
- 전체 강사를 점수순으로 반환 (필터링 없음)
- 점수 0점인 강사도 목록에 포함 (UI에서 표시 여부 결정)

### BR-M08: 매칭 근거 생성 규칙
- 강점: 항목 점수가 해당 항목 최대 점수의 70% 이상
- 약점: 항목 점수가 해당 항목 최대 점수의 30% 미만
- 요약 문장: 최대 100자
- 매칭된 키워드: 실제 일치한 키워드만 표시

## 에러 처리 규칙

### BR-E01: 파서 에러
| 에러 | 원인 | 처리 |
|---|---|---|
| UnsupportedFileTypeError | 지원하지 않는 형식 | 400 에러, 명확한 메시지 |
| ParseError | 파싱 실패 (손상/보호) | 400 에러, 원인 설명 |
| EmptyDocumentError | 빈 문서 | 400 에러, 안내 메시지 |
| SectionNotFoundError | 섹션 미식별 | 전체 텍스트 반환 + 경고 |

### BR-E02: 매칭 에러
| 에러 | 원인 | 처리 |
|---|---|---|
| NoInstructorsError | 강사 풀이 비어있음 | 빈 결과 반환 + 경고 |
| InvalidRequirementsError | 요구사항이 비어있음 | 400 에러, 파싱 재시도 안내 |

## 검증 규칙

### BR-V01: 입력 검증
- 파일: null 불가, 크기 > 0, 크기 <= 50MB
- Instructor 리스트: null 불가 (빈 리스트는 허용)
- TaskRequirements: qualifications 또는 evaluation_criteria 중 하나는 있어야 함
